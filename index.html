Young Money


Front-End/React Developer

Young Money is a web application that was developed to help parents manage their children's allowances and expenses while motivating them to do house work and chores.


We built out the front-end using React-Materialize to provide a dynamic interface that allowed for stateful changes on key components. Other front-end technologies used include Jquery, ES6/ Vanilla JS.

Overlord


Lead Front-End Developer

The front-end was built out using Bootstrap, Jquery, Javascript, Handlebars, & Vanilla CSS.
